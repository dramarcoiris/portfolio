# Portfolio personal

Portfolio desarrollado para presentar mi trabajo como desarrolladora Full Stack y diseñadora gráfica.

Este proyecto nace con el objetivo de reunir en un único espacio los proyectos, tecnologías y experiencias que forman parte de mi trayectoria profesional. Una web diseñada y desarrollada desde cero para reflejar mi forma de entender el desarrollo: funcional, visual y centrado en las personas.

## Tecnologías

- React
- TypeScript
- React Router
- Tailwind CSS
- Vite

## Contenido

El portfolio incluye:

- Presentación personal.
- Información sobre mi trayectoria y perfil profesional.
- Tecnologías y herramientas con las que trabajo.
- Proyectos destacados con páginas de detalle individuales.
- Información de contacto y enlaces profesionales.
